# ✅ 기능 구현 체크리스트 (Feature Checklist)

현재 프로젝트인 **직장인 소모임/커뮤니티 서비스**의 구현된 기능 목록입니다.

## 1. 초기 셋업 (Initialization)
- [x] 서비스 기획 및 MVP 정의
- [x] 기술 스택 선정 (Next.js, Supabase, Tailwind, Leaflet)
- [x] 데이터베이스 스키마 설계 (PostgreSQL)
- [x] 프로젝트 초기화 및 환경 설정

## 2. 인증 및 사용자 (Auth & User)
- [x] 이메일 회원가입 기능 (Email Sign Up)
- [x] 이메일 로그인 기능 (Email Sign In)
- [x] 유저 프로필 관리 (User Profile)
    - 닉네임, 프로필 이미지 표시 (Gravatar/Dicebear 연동)

## 3. 모임 관리 (Gatherings)
- [x] **모임 개설 (Create):** 제목, 내용, 일시, 장소, 카테고리, 이미지 등록
- [x] **위치 기반 장소 찾기:** 카카오 우편번호 서비스 연동
- [x] **모임 상세 보기:** 참여 현황, 호스트 정보, 지도 확인
- [x] **모임 삭제:** 호스트 권한 확인 후 모임 삭제 기능
- [x] **지난 모임 처리:** 종료된 모임 자동 블러 처리 및 참여 불가 설정

## 4. 위치 기반 서비스 (Location Based Service)
- [x] **좌표 변환:** 주소를 위도/경도로 자동 변환 저장
- [x] **내 주변 찾기:** 현재 위치 기준으로 가까운 모임 정렬 (거리 표시)
- [x] **지도로 보기:** Leaflet 지도를 통해 전체 모임 위치 시각화

## 5. 커뮤니티 (Community)
- [x] **참여 신청:** 모임에 참여하기 버튼 클릭 시 즉시 참여
- [x] **댓글 기능:** 문의사항이나 기대평을 남길 수 있는 댓글 시스템
- [x] **찜하기(북마크):** 관심 있는 모임을 저장하는 기능

## 6. 마이페이지 (My Page)
- [x] **대시보드:** `/mypage` 경로에서 통합 관리
- [x] **탭(Tab) UI:**
    - 👤 **내가 만든 모임:** Host로서 개설한 모임 목록
    - 📅 **참여한 모임:** Guest로서 참여한 모임 목록
    - ❤️ **찜한 모임:** 관심 등록한 모임 목록

## 7. 검색 및 탐색 (Search & Discovery)
- [x] **통합 검색:** 제목 및 내용 키워드 검색
- [x] **카테고리 필터:** 주제별(운동, 스터디 등) 모아보기
- [x] **상태 필터:** '모집 중'인 모임만 보기 토글

---
*모든 기능은 Next.js App Router와 Server Actions를 사용하여 최신 방식으로 구현되었습니다.*
