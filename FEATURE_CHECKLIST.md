# ✅ 기능 구현 체크리스트 (Feature Checklist)

현재 프로젝트인 **직장인 소모임/커뮤니티 서비스**의 구현된 기능 목록입니다.

## 1. 인증 및 사용자 (User Management)
- [x] **회원가입:** 이메일 주소로 회원가입
- [x] **로그인:** 이메일/비밀번호 로그인
- [x] **프로필:** 닉네임 및 사용자 아이콘 표시

## 2. 모임 관리 (Gatherings)
- [x] **모임 개설:** 제목, 내용, 일시, 장소, 카테고리, 이미지 등록
- [x] **장소 찾기:** 주소 검색을 통한 정확한 모임 장소 지정
- [x] **모임 상세:** 참여 현황, 호스트, 지도 위치 확인
- [x] **모임 삭제:** 개설자(호스트)의 모임 삭제 기능
- [x] **종료 모임:** 날짜가 지난 모임은 자동으로 '종료됨' 표시
- [x] **승인 시스템:** 호스트가 참여 신청을 승인/거절하여 인원 관리

## 3. 위치 기반 서비스 (Location Features)
- [x] **내 주변 찾기:** 현재 위치 기준으로 가까운 모임 정렬 (거리 표시)
- [x] **지도 보기:** 전체 모임의 위치를 지도 상에서 한눈에 확인
- [x] **거리 계산:** 내 위치와 모임 장소 간의 직선 거리 표시

## 4. 커뮤니티 (Community)
- [x] **참여 신청:** 승인제 모임은 호스트 승인 후 참여 확정
- [x] **신청 관리:** 대기 중인 참여 신청 취소 기능
- [x] **댓글:** 모임에 대한 문의나 소통을 위한 댓글 작성/삭제
- [x] **찜하기:** 관심 있는 모임 저장 (북마크)

## 5. 마이페이지 (My Page)
- [x] **통합 대시보드:** 나의 모임 활동 내역을 한곳에서 관리
    - **내가 만든 모임:** 직접 개설한 모임 관리
    - **참여한 모임:** 참여 신청한 모임 목록
    - **찜한 모임:** 관심 등록한 모임 목록

## 6. 검색 및 탐색 (Search & Discovery)
- [x] **키워드 검색:** 모임 제목 및 내용으로 검색
- [x] **카테고리 필터:** 관심사별(운동, 스터디 등) 모아보기
- [x] **모집 상태 필터:** 모집 중인 모임만 골라보기
